How to use this service?

1. Go to Freshservice -> Admin -> Alert Profiles.

2. Create an Alert Profile and Custom Integration with the help of this solution article (don't change the default payload).

3. In the Custom Integration created, copy the Endpoint URL and Auth header under Authorize Integration section.

4. Paste the copied information to this form and Send Alerts.

5. This will take 2-3 minutes to create alerts in the demo account.

6. This will generate 100-120 alerts along with some alert events.

7. In case alerts are not created in the demo instance or for any queries contact the developer.
